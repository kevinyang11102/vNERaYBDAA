# 河长媒资管理系统 java148

## 项目概述

河长媒资管理系统是一款基于Spring Boot和MyBatis技术栈，前端采用Layui与HTML进行开发的媒资管理平台。该系统旨在为河长制管理人员提供便捷的媒资管理服务。

## 技术栈

### 后端

- Spring Boot
- MyBatis-Plus

### 前端

- Layui
- HTML

## 运行环境

- 开发工具：IDEA
- 数据库：MySQL 5.7
- 运行方式：Spring Boot 直接运行

## 功能模块

系统主要包括以下功能模块：

- 媒资管理
- 用户权限控制
- 数据统计与分析

## 系统角色

- 超级管理员
- 普通用户

## 目录结构

```
河长媒资管理系统
├── src
│   ├── main
│   │   ├── java
│   │   ├── resources
│   │   └── webapp
│   └── test
└── pom.xml
```

## 核心亮点

- 基于Spring Boot，易于部署与维护。
- 使用MyBatis-Plus简化数据库开发。
- Layui与HTML结合的前端设计，界面简洁大方。

## 部署步骤

1. 确保MySQL 5.7数据库已正确安装并运行。
2. 导入数据库文件，建立所需的数据表。
3. 使用IDEA或相应工具导入项目，并编译。
4. 运行Spring Boot应用程序。
5. 访问 `http://localhost:8080/loginShiro` 进行登录。
